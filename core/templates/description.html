О нас

Сайт предназначен для ознакомительных целей.

Веб-приложение помогает предсказать исход матча основываясь на алгоритме, который анализиурет прошедшие матчи и статистику.

Сайт разрабатывается для использования в практическом и лабораторном курсе дисциплин кафедры ИСП. Сайт предоставляет только самые необходимые функции для среднестатистического пользователя.

Cайт был спроектирован и разработан Диденко Данилом и Панченко Павлом.
